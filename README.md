# Charity-Donation-Tracker
Charity Donation Tracker is a decentralized application (DApp) that promotes transparency in charitable giving by recording transactions on the Ethereum Sepolia test network. Users can connect their wallets, donate Sepolia ETH, view the total amount raised, track the campaign’s progress and see the donation history. The application also allows the administrator to securely withdraw funds while maintaining transparency through on-chain records.
